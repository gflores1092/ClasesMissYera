# Comunicado institucional

## Overview
Genera comunicados institucionales que suenan profesionales y cercanos a la vez, con la estructura completa y sin datos inventados. Pensado para difusión interna o externa.

## Cuándo usar este skill
- El usuario pide un "comunicado", "anuncio", "aviso", "circular" o "nota para difundir".
- Necesita informar de un cambio, un evento, una fecha, un proceso o una novedad.

## Procedimiento
1. Identifica: **a quién** se dirige, **qué** se anuncia, **por qué** importa, **qué debe hacer** el lector y **cuándo**.
2. Si algún dato clave falta (fecha, lugar, enlace, contacto), márcalo como `[POR COMPLETAR]`.
3. Redacta en el formato de salida, en tono formal pero humano.

## Formato de salida
- **Asunto / título:** claro y específico.
- **Saludo:** dirigido al público correcto.
- **Cuerpo:** 1-3 párrafos cortos. Primero lo más importante (qué y cuándo), luego el detalle.
- **Llamado a la acción:** qué debe hacer el lector y para cuándo.
- **Cierre:** una línea de contacto o referencia.
- **Firma:** [Área / responsable] · [Fecha]

## Reglas
- **No inventes** fechas, nombres, enlaces ni cifras: usa `[POR COMPLETAR]`.
- Lenguaje claro y directo; evita jerga y palabras de relleno.
- Frases y párrafos cortos. Máximo ~250 palabras salvo que se pida más.
- Tono respetuoso y cercano, nunca frío ni burocrático.
- Una sola idea central por comunicado.
